# BiddBuddies  
A full web platform for trading items and running auctions, built with **PHP**, **MySQL**, **HTML**, **CSS**, **JavaScript**, and **AJAX**.

## Features  
✅ User registration & authentication  
✅ Post items for trade or auction  
✅ Real-time bidding system (AJAX updates)  
✅ Favorites system  
✅ Account settings page  
✅ Admin panel to approve auctions and manage users  
✅ Fully responsive frontend  

## Technologies Used  
- PHP (Backend)  
- MySQL (Database)  
- HTML, CSS, JavaScript (Frontend)  
- AJAX (Real-time updates)  

## Notes  
⚠️ Database credential files (connection.php) were removed for security.
